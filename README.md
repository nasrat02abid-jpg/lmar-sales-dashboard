# 🏠 Lmar Marketing — Sales Performance Dashboard 2025

An interactive real estate sales dashboard built with Python & Chart.js

## 🔗 Live Demo
[View Dashboard](https://nasrat02abid-jpg.github.io/lmar-sales-dashboard)

## 📊 Features
- 5 KPI Cards (Revenue, Units, Commission, Conversion)
- Monthly Revenue & Units trend
- Agent Leaderboard
- Society & City performance
- Lead Source analysis

## 🛠 Built With
- Python (Pandas, NumPy) — data processing
- Chart.js — interactive visualizations
- HTML/CSS — dashboard layout
